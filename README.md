# Keylogger 

## Description
This project demonstrates the working principle of a keylogger using Python.
It captures keystrokes typed inside a GUI window and stores them in
a text file and JSON file.

⚠️ This project is strictly for educational purposes and does not capture
system-wide keystrokes.

## Technologies Used
- Python
- Tkinter
- JSON

## Files
- keylogger.py – main program
- keystrokes.txt – text log file
- keystrokes.json – JSON log file

## How to Run
1. Install Python
2. Run `python keylogger.py`
3. Type inside the GUI window

## Output
Keystrokes are stored with date and time in both text and JSON files.
